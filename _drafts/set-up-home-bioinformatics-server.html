WARNING: it should go without saying, but I will say it nonetheless. If you value anything in your current set-up, stop now. If you continue do a back-up before starting. Better yet. When was the last time you backed-up your files? Do it again now regardless.

Download iso

The iso can be found in debian-8.3.0-amd64-netinst.iso, and I choose the latest (minimal) version

During the installation there wher issue with properitary drivers, so I had to donwload a version that includes these.

create liveUSB

I used this tutoriall because I am using Ubuntu. Unibotin is also a good option for other OS.

Install Debian from USB

To boot from the USB press F12 when the computer starts, and choose USB in the boot menu options. Then I followed the instructions here (graphical install). I choose English (UK) as the language, Germany as a location, and Portuguese keyboard layout (yes, that is weird).

graphical interface

Xfce is a lightweight interface useful for under-powered and old computers. Install with:

sudo apt-get update &&
sudo apt-get install -y xfce4 xfce4-goodies

To start: sudo startxfce4. sudo is not required but it might useful to mount external drives. Exit by logging out.

reboot from command-line

sudo reboot
